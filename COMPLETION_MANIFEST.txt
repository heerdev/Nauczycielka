API ENDPOINTS
================================================================================

Courses:
  POST   /api/v1/courses                          - Create course
  GET    /api/v1/courses                          - Get all courses
  GET    /api/v1/courses/{courseId}               - Get course by ID
  GET    /api/v1/courses/instructor/{instructorId} - Get courses by instructor
  PUT    /api/v1/courses/{courseId}               - Update course
  DELETE /api/v1/courses/{courseId}               - Delete course

Live Sessions:
  POST   /api/v1/courses/{courseId}/sessions                    - Start session
  GET    /api/v1/courses/{courseId}/sessions                    - Get sessions
  GET    /api/v1/courses/{courseId}/sessions/{sessionId}        - Get session
  POST   /api/v1/courses/{courseId}/sessions/{sessionId}/transcripts - Add transcript
  POST   /api/v1/courses/{courseId}/sessions/{sessionId}/end    - End session (trigger AI)

Vocabulary:
  GET    /api/v1/courses/{courseId}/vocabulary                  - Get vocabulary
  GET    /api/v1/courses/{courseId}/vocabulary/student/{studentId} - Get student vocabulary
  GET    /api/v1/courses/{courseId}/vocabulary/{itemId}         - Get vocabulary item

Summaries:
  GET    /api/v1/courses/{courseId}/summary                     - Get course summary
  GET    /api/v1/courses/{courseId}/summary/session/{sessionId} - Get session summary

Enrollments:
  POST   /api/v1/courses/{courseId}/enrollments                 - Enroll student
  GET    /api/v1/courses/{courseId}/enrollments                 - Get enrollments
  DELETE /api/v1/courses/{courseId}/enrollments/student/{studentId} - Remove student

Health & Monitoring:
  GET    /actuator/health                         - Health check
  GET    /actuator/info                           - App info
  GET    /actuator/metrics                        - Metrics

RUNNING THE APPLICATION
================================================================================

Prerequisites:
  ✅ Java 21+
  ✅ Maven 3.9+
  ✅ Ollama installed
  ✅ Qwen2 model pulled

Quick Start:
  1. ollama serve                          (In separate terminal)
  2. cd /Users/zuahmed/Project/Nauczycielka
  3. mvn clean install
  4. mvn spring-boot:run

Access Points:
  • Application: http://localhost:8080
  • API: http://localhost:8080/api/v1
  • Health: http://localhost:8080/actuator/health
  • H2 Console: http://localhost:8080/h2-console
  • Ollama: http://localhost:11434

Docker Deployment:
  1. mvn clean package
  2. docker-compose up -d
